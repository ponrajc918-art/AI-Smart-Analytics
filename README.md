# AI-Powered Smart Data Analytics and Prediction System

## Overview

This project is a web-based application developed using Streamlit that performs data analysis and prediction using machine learning models. The system helps in making data-driven decisions by analyzing input data and generating predictions with visual insights.

## Features

* Student Performance Prediction
* Loan Approval Prediction System
* Staff Performance Analysis
* Data Visualization using charts and graphs
* Model Explainability using SHAP

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib
* Seaborn
* Plotly

## Machine Learning Models

* Linear Regression
* Decision Tree
* Classification Algorithms

## Dataset

The dataset includes:

* Student academic records
* Loan applicant details
* Staff performance data
